# Ejemplos

El proyecto dbt de ejemplo (con anti-patrones sembrados) vive dentro del paquete para que
`datanomad-review demo` funcione desde cualquier instalación:

→ [`src/datanomad_review/examples/sample-dbt-project/`](../src/datanomad_review/examples/sample-dbt-project/)

Pruébalo sin clonar nada:

```bash
pip install datanomad-review
datanomad-review demo
```
